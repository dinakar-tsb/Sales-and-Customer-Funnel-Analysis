# Power BI Sales & Customer Funnel Analysis

## Project Overview
This project presents an interactive **Sales & Customer Funnel Analysis Dashboard** built using **Power BI**.

The dashboard helps analyze:
- Sales performance
- Customer behaviour
- Conversion funnel
- Profit trends
- Regional performance

The goal of this project is to transform raw sales data into actionable business insights using data visualization.

---

## Objectives
- Monitor overall sales performance
- Track customer funnel stages
- Identify high-performing products and regions
- Analyze profit vs sales trends
- Improve business decision-making using data

---

## Tools & Technologies
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX Measures
- Data Modeling
- Excel / CSV dataset
